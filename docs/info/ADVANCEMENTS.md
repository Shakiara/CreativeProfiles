# Advancements

Creative Profiles separates advancements between Survival and Creative profiles.

## Behavior

When switching profiles, the mod saves the current profile's advancement file and loads the target profile's advancement file.

If the target profile has no saved advancements yet, it starts with an empty advancement state.

## Backups

Manual and automatic backups include advancement data.

Admin restore commands also restore advancement data for the selected profile.

## Storage

Advancement profile files are stored in the world's advancements directory next to the normal player advancement file.

They are managed by Creative Profiles and should not be edited while the server is running.
