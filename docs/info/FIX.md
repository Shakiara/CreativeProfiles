# Fix Command

## Command

```text
/cp fix
```

## Permission

Requires permission level 2.

## Description

Repairs the player's current profile marker based on the dimension they are currently in.

If the player is in the configured Creative world, the profile is set to `creative`. Otherwise, it is set to `survival`.

## Safety

This command does not delete inventories, XP, Ender Chests, positions, advancements, optional mod data, or backup data.

Use `/cp status` after running this command to confirm the result.
