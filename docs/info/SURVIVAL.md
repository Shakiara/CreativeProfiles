# Survival Command

## Command

```text
/survival
```

## Permission

Requires permission level 2.

## Description

Switches the player to the Survival profile.

Before switching, Creative Profiles creates an automatic backup and saves the current Creative profile data. It then loads the Survival profile data and teleports the player to the configured Survival world.

## Default Survival World

```text
minecraft:overworld
```

The configured Survival world must exist.
