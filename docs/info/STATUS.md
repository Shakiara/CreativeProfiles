# Status Command

## Commands

```text
/cp status
/cp status <player>
```

## Permission

`/cp status` requires permission level 2.

`/cp status <player>` requires permission level 4.

## Description

Shows profile state and diagnostic information.

The output includes:

- Current profile
- Current dimension
- Configured Creative world
- Configured Survival world
- Whether the current profile matches the current dimension
- Saved data for Survival and Creative profiles
- Manual and automatic backup status
- Curios load state
- Cosmetic Armor Reworked load state

Use this command after switching, backing up, restoring, or repairing profile data.
