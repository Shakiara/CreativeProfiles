# Swap Data Command

## Command

```text
/cp swapdata <player>
```

## Permission

Requires permission level 4.

## Description

Swaps stored Survival and Creative profile data for a player.

This is an administrative repair tool for cases where profile data was crossed and Survival data ended up saved as Creative data, or Creative data ended up saved as Survival data.

## Swapped Data

- Inventory
- XP
- Ender Chest
- Position
- Advancements
- Curios data
- Cosmetic Armor Reworked data

## Notes

This command does not change the player's current profile marker. It only swaps the saved profile data and then reapplies the active profile data to the player.
