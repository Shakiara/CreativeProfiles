# Creative Command

## Command

```text
/creative
```

## Permission

Requires permission level 2.

## Description

Switches the player to the Creative profile.

Before switching, Creative Profiles creates an automatic backup and saves the current Survival profile data. It then loads the Creative profile data and teleports the player to the configured Creative world.

If no Creative inventory exists yet, the player starts with an empty Creative inventory.

## Default Creative World

```text
creative
```

The configured Creative world must already exist.
