# Code of Conduct

## Our Pledge

We want Creative Profiles to be a welcoming project for players, server owners, modpack creators, and contributors.

Everyone participating in the project is expected to be respectful, patient, and constructive.

## Expected Behavior

- Be kind and professional.
- Explain problems clearly.
- Respect different levels of experience.
- Keep feedback focused on the project and the code.
- Help keep discussions useful for everyone.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Repeated off-topic disruption.
- Sharing private information without permission.
- Abusive or intentionally hostile communication.

## Enforcement

Project maintainers may remove comments, close issues, or block users when behavior is harmful to the project or its community.

If you need to report a conduct issue, contact the project maintainer privately when possible.

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, releases, and community conversations directly related to Creative Profiles.
